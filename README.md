# afeefa-frontend
The main web app of Afeefa.de

## Setup

### set API source
change path in *APPAFEEFA.js*:
`
...
that.setConfig(
	{
		apiUrl: 'http://api.afeefa.fx/',
		...
	}
...
`
### Run the app
- set up a virtual host or something, which points at the root folder
- open your virtual host's domain and it should run
